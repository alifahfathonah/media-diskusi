# Media Diskusi

## Configurasi Project

```
1. import database yang berada pada folder sql
2. ubah base url pada file --> application/config/config.php
3. ubah configurasi database pada file database.php --> application/config/database.php
4. ubah variabel base url pada file menu.js --> assets/js/admin/menu.js
5. ubah variabel base url pada file role.js --> assets/js/admin/role.js
```

### Catatan

```
1. file role.js & menu.js sewaktu-waktu akan berubah seiring project dikerjakan dan diupdate
2. sangat baik jika dijalankan pada browser Firefox Web Browser
```
